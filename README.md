# FizzBuzz

## What is This Project?
This program prints out numbers 1 to 100. However, if the number is divisble by 3 it prints out Fuzz. If the number is divisible by 5 it prints out Buzz and if the number is divisible by 3 and 5 it prints out FIzzBuzz. 

## How Does it Work?
The program uses a for loop to print out the numbers 1 - 100. In the for loop the program uses if statements to check if the number is **divisible** by:
 - 3 & 5
 - just 5 
 - just 3 

If the number is divisible by **3 & 5**: 

--> the program prints out the phrase **FizzBuzz** instead of the number.

If the number is divisible by **5**:

--> the program prints out the word **Buzz** instead of the number.

If the number is divisible by **3**:

--> the program prints out the word **Fizz** instead of the number.

## Order Matters!!!
 You want to write the program in that order specifically. This is because you want the program to check the broader categories before the more specific ones, so it doesn't skip over the broader categories altogether.
